# 🧠 Database Design Practice (ERD)

This repository contains my database design practice, where I model real-world systems using ER diagrams (ERDs).

The goal is to improve my understanding of how scalable applications are structured at the database level.

---

- ER diagrams for different real-world use cases  
- Well-structured relational models  
- Use of proper relationships (1:1, 1:N, N:M)  
- Handling of subscriptions, sessions, and tracking systems  
- Practical database thinking beyond basic CRUD
